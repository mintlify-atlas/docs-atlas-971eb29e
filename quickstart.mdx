# Quick Start Guide

This guide will help you install and start using auto-cpufreq to optimize your Linux laptop's battery life and performance.

## Prerequisites

Before installing auto-cpufreq, ensure you have:

- A Linux system with Intel, AMD, or ARM CPU
- `sudo` access
- Git installed
- Python 3.9 or higher

## Installation

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/AdnanHodzic/auto-cpufreq.git
    cd auto-cpufreq
    ```
  </Step>
  
  <Step title="Run the installer">
    ```bash
    sudo ./auto-cpufreq-installer
    ```
    
    The installer will:
    - Install auto-cpufreq and its dependencies
    - Set up the binary in `/usr/local/bin/`
    - Configure the system for optimal operation
  </Step>
  
  <Step title="Test with monitor mode">
    Before installing the daemon, preview what auto-cpufreq will do:
    
    ```bash
    sudo auto-cpufreq --monitor
    ```
    
    This shows optimization suggestions without making any changes.
  </Step>
  
  <Step title="Install the daemon">
    Once you're satisfied with the preview, install the daemon for permanent optimizations:
    
    ```bash
    sudo auto-cpufreq --install
    ```
    
    The daemon will now:
    - Start automatically on boot
    - Continuously optimize CPU performance and power usage
    - Adapt to battery vs AC power state
  </Step>
</Steps>

## Verify Installation

Check that the daemon is running:

```bash
# Check service status
systemctl status auto-cpufreq

# View live statistics
auto-cpufreq --stats
```

## Common First Steps

### 1. Check Current Status

```bash
auto-cpufreq --stats
```

This shows:
- Current CPU frequency and governor
- Battery state
- Temperature
- What optimizations are being applied

### 2. Configure for Your Needs

Create a configuration file at `/etc/auto-cpufreq.conf`:

```ini
[charger]
governor = performance
turbo = auto

[battery]
governor = powersave
turbo = auto
```

Then restart the daemon:

```bash
sudo systemctl restart auto-cpufreq
```

### 3. Monitor Battery Life

Track your battery usage over a few days to see the improvement. Most users report 20-30% longer battery life.

## Troubleshooting

<AccordionGroup>
  <Accordion title="Service won't start">
    Check for conflicting services:
    ```bash
    systemctl status power-profiles-daemon
    systemctl status tlp
    ```
    
    Disable them:
    ```bash
    sudo systemctl mask power-profiles-daemon
    sudo systemctl disable --now tlp
    ```
  </Accordion>
  
  <Accordion title="CPU not scaling properly">
    Try switching CPU drivers. Edit `/etc/default/grub`:
    ```
    GRUB_CMDLINE_LINUX_DEFAULT="quiet splash intel_pstate=disable"
    ```
    
    Then:
    ```bash
    sudo update-grub
    sudo reboot
    ```
  </Accordion>
  
  <Accordion title="High temperatures">
    Check if turbo boost is disabled on battery:
    ```bash
    auto-cpufreq --stats
    ```
    
    If needed, force turbo off:
    ```bash
    sudo auto-cpufreq --turbo=never
    ```
  </Accordion>
</AccordionGroup>
